# Getting Started with Church Crypto Donations

Welcome to your church's cryptocurrency donation platform! This guide will help you get started with accepting crypto donations.

## What You'll Learn

In this section, you'll learn how to:

- Create your first crypto wallet
- Import an existing wallet
- Generate QR codes for donations
- Monitor incoming donations
- Understand wallet security

## Quick Start Checklist

1. ✅ Create your first wallet (BTC or USDC)
2. ✅ Save your seed phrase securely
3. ✅ Generate a QR code
4. ✅ Share with your donors
5. ✅ Monitor balances on the dashboard
