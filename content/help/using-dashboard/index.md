# Using the Dashboard

Your dashboard is command central for managing crypto donations. This guide covers all features.

## Dashboard Overview

The dashboard shows:

- **Total Balance** - Combined value of all wallets in USD
- **Active Wallets** - Number of wallets you've created
- **Total Donations** - Number of donation transactions received
- **Quick Actions** - Links to common tasks

## Main Features

### Wallet Management
- Create new wallets
- Import existing wallets
- View wallet balances
- Generate QR codes
- Swap between cryptocurrencies
- Withdraw to bank account

### Monitoring
- Live balance updates (every 10 minutes)
- USD conversion rates
- Transaction notifications (coming soon)

### Security
- Multi-user access with roles
- Audit logs
- Non-custodial wallet system
